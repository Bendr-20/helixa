# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| V2 (HelixaV2.sol) | ✅ Active |
| V1 (AgentDNA.sol) | ❌ Deprecated |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report security issues to: **security@helixa.xyz**

Or DM [@BendrAI_eth](https://x.com/BendrAI_eth) on X.

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

- Smart contracts in `src/`
- API server in `api/`
- Frontend in `frontend-v2/`

## Bug Bounty

We don't have a formal bug bounty program yet, but we appreciate responsible disclosure and will credit researchers who report valid vulnerabilities.
