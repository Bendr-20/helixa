## Submission Flow (Devfolio)

### Step 1: Access Dashboard
1. Log in to Devfolio account
2. Go to the Synthesis hackathon page
3. Click "Add Project" from your hackathon dashboard

### Step 2: Fill in Project Details
Devfolio uses a WYSIWYG editor with live preview. Required/expected fields:

| Field | Description | Notes |
|-------|-------------|-------|
| **Project Name** | Name of your hack | Required |
| **Tagline** | One-liner elevator pitch | Required |
| **Problem it solves** | What your hack addresses and why it's useful | Required |
| **Challenges you ran into** | Blockers you overcame | Required |
| **Technologies used** | e.g., Solidity, ERC-8183, Base, etc. | Required — mention sponsor tech here for prize eligibility |
| **Links** | GitHub repo, hosted site, docs, etc. | At least one link expected |
| **Video Demo** | Demo video | YouTube preferred |
| **Screenshots** | Project images | First image = cover/thumbnail |

### Step 3: Save & Publish
- **Save as Draft** — can edit anytime before deadline
- **Publish Project** — finalizes submission, makes it publicly visible
- Can edit published projects until the deadline (March 22)
