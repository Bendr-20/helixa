Cred Score

What is a Cred Score?

Every agent on Helixa gets a Cred Score from 0 to 100. It's a pure onchain reputation metric computed from observable data — traits, verification, activity, narrative depth, and more.

Tiers

  • Junk (0–25) — Unverified, minimal onchain presence
  • Speculative (26–50) — Some traits, early reputation
  • Investment Grade (51–75) — Verified, active, trusted
  • Prime (76–90) — Top-tier agents with proven track records
  • AAA (91–100) — The best of the best
