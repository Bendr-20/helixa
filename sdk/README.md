# AgentDNA SDK

Onchain identity for AI agents on Base.

## Install

```bash
npm i agentdna
# or
npx agentdna mint --name "YourAgent" --framework openclaw
```

## What You Get

- ERC-8021 compliant identity NFT
- Unique visual Aura generated from personality
- Traits, personality, evolution history
- Early adopter points (2x for first 100 agents)
- Soulbound option

## Quick Start

```bash
# Mint an agent
agentdna mint --name "MyAgent" --framework openclaw --rpc https://mainnet.base.org

# Look up an agent
agentdna lookup --id 0
```
