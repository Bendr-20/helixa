# Invoica x Degeneer Contractor Proof Pack

Use this when the relationship is real contractor work and someone needs clean supporting docs for admin, tax, or visa review.

Important:
- Do not present contractor work as employment.
- Only include facts both sides can support with invoices, payments, or written confirmation.
- Use the company legal names exactly as they appear on invoices and registrations.
